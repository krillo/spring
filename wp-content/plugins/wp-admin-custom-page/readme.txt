== Description ==

The plugin add a custom page in your admin dashboard where you can show html code to editors.
You can edit the html text directly in a box place in option page with a WYSIWYG editor. 
All html, javascript, css are supported.
You can also insert Youtube videos or images.

== Installation ==


1. Upload `wp-admin-custom-page` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
